#  openLibreReader - iOS

openLibreReader for iOS is a project to connect the various Libre Transmitters to the iPhone.

## Features:
- supported devices:
    - blueReader
    - nightscout client
    - LimiTTer
- configurable Alarms
- configurable Notifications
- supports mg/dl and mmol units
- push to nightscout
- todayWidget
- simple watch app
- simple linear regression calibration
